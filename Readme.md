# Web in Rust

Simple web in RUST that uses Actix-web and Tera to render HTML templates.

## Installation

**Prerequisites**

- Rust
- Make

**Clone the repository**

```bash
git clone https://github.com/DrxcoDev2/rust-web-hello-world.git
```
**Make a `.env`file**

**Build the project**

```bash
make build
```

**Run the project**

```bash
make run
```

**Clean the project**

```bash
make clean
```

## Generate Template
```bash
chmod +x rust-template.sh
./rust-template.sh
```
